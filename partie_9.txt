Supabase permet d'écouter les changements en temps réel. Si le développeur n'a pas configuré les politiques de réplication, tu peux écouter tout ce qui se passe sur le site : nouvelles connexions, messages privés, nouveaux scripts, etc.
JavaScript
const channel = mySupabase
  .channel('public:profiles')
  .on('postgres_changes', { event: '*', schema: 'public', table: 'profiles' }, payload => {
    console.log('CHANGEMENT DÉTECTÉ SUR UN PROFIL !', payload);
  })
  .subscribe();


console.log("Écoute Realtime activée. Attends qu'un utilisateur change un truc...");
